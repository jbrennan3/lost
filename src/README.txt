The files in this directory implement a very simple web application to take in and present user entered data.

This is very much a work in progress.

Files:
app.py - A Flask app to be run via mod_wsgi
templates/
	index.html - main login window
	main.html - main user report screen
	asset_report.html - generated asset report based on user filters
	transit_report.html - generated transit report based on user filters
	logout_page.html - log off screen with redirect back to index
